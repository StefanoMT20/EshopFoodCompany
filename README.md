# EshopFoodCompany
This a webpage of a company that sells food in USA with annual membership.


It's full responsive website.
I used the following tools:

- HTML 5(BEM)
- CSS 3
- Grid (layout) & Flexbox (blocks)

## Installation
```sh
pull or fork the repository
open index.html
```
